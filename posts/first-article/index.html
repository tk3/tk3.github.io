        tk3.github.io を作成してから初めてのブログ。

        以前からブログのようにその日に書き残すようなものではなく、文章を書きためたり、その文書を更新していけるようにしたいと考えていた。ブログでも過去の記事を更新すれば良さそうだが、それではブログぽさがない。やはり文書を残すサイトという感じにしたかった。

        ちなみにこのサイトは GitHub Actions で Github Pages にデプロイしている。 どこにホスティングしようか迷ったが、簡単にできる Github Pages を選択した。もしかしたら、 tk3.dev の方に移すかもしれない。
